Exemplo 26 CSS - Propriedades de Fontes e Textos

Fontes

Quando aplicamos folhas estilo em documentos html, procuramos estilizar textos que podem ser estilizados de formas diferentes pois podemos mudar as diversas propriedades das fontes e do próprio texto. As propriedadeS de fonts, incluem cores, peso, estilo, tamanho, dentre outros valores. Neste exemplo, estamos usando propriedades de fonts diferentes.

Valores possíveis aplicados a propriedade font em css

  • font-family → Define o tipo de fonte (ex: Arial, Roboto, serif)
  • font-size → Define o tamanho da fonte
  • font-weight → Define o peso da fonte (normal, bold, 100–900)
  • font-style → Define o estilo (normal, italic, oblique)
  • font-variant → Controla variações como small-caps
  • font-stretch → Ajusta a largura da fonte (condensada ou expandida)
  • font-size-adjust → Ajusta a legibilidade com base na altura-x
  • font-kerning → Controla o espaçamento entre pares de letras
  • font-feature-settings → Ativa recursos tipográficos avançados (OpenType)
  • font-variation-settings → Controla fontes variáveis (variable fonts)
  • font → Propriedade resumida (atalho para várias configurações)

Propriedade Texto

As propriedades de texto em CSS são usadas para controlar a aparência, alinhamento e legibilidade do conteúdo textual. Essas propriedades são amplamente aplicadas aos documentos HTML e elas não alteram a fonte em si,mas elas influenciam a forma como o texto será exibido e organizado visualmente no documento html.

Princípais valores aplicados a propriedade texto

  • text-align → Define o alinhamento do texto (left, right, center, justify)
  • text-transform → Controla a transformação do texto (uppercase, lowercase, capitalize)
  • text-decoration → Adiciona decoração ao texto (underline, line-through, overline)
  • text-indent → Define o recuo da primeira linha do parágrafo
  • letter-spacing → Controla o espaçamento entre letras
  • word-spacing → Controla o espaçamento entre palavras
  • line-height → Define o espaçamento entre linhas
  • text-shadow → Adiciona sombra ao texto
  • white-space → Controla como espaços e quebras de linha são tratados
  • overflow-wrap → Define como palavras longas quebram linha
