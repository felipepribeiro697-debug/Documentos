Exemplo 4 HTML - Estrutura Geral

Este Exemplo mostra a estrutura mínima de um documento html. A seguir, são descritas as principais tags estruturantes de um documento html.

  • <DOCTYPE html>:É uma instrução html, essa tag define tipo/versão do html que será usado no documento html.
  • <html>:É o elemento central do documento, é o container do documento html ela engloba toda a página.
  • <head>:Essa tag contém os metadados que serão interpretados pelo navegador, tais como metadados de codificação. Nela também temos a tag title que define o título do documento.
  • <body>:Este bloco é responsável por deixar visível o conteúdo destinados aos usuários. Ou seja, todo o conteúdo do documento html.
