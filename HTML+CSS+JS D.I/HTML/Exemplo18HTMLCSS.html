Seletor de Classe

O seletor de classe CSS é uma forma de aplicar estilos a elementos HTML específicos, identificados por um atributo class. Ele é definido por um ponto (.) seguido do nome da classe, permitindo reusabilidade, pois múltiplos elementos podem compartilhar a mesma classe.

Neste exemplo estamos usando vários seletores de classe. Cada seletor de classe começará sempre com um (.), seguido do nome do seletor {propriedade:valor}
