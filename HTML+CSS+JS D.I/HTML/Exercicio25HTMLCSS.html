Exemplo 25 HTML CSS - Vinculação Interna

A vinculação Interna de arquivos CSS pode se dar de duas formas: Incorporada e Inline. A vinculação interna incorporada é aquela onde criamos as regras CSS dentro da Tag Style, geralmente dentro da Tag Head do documento HTML. Já a vinculação Inline é aquela onde se coloca a formatação diretamente no elemento HTML.

Vinculação Inline

A vinculação interna - inline é a menos recomendada, isso ocorre pois ao usá-la teremos uma série de problemas de manutenibilidade e reaproveitamento. Esse tipo de vinculação só é recomendado em casos muito específicos e geralmente em testes. Neste exemplo, estamos usando a vinculação inline.
