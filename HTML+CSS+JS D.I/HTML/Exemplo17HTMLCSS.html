Exemplo 17 - HTML CSS

CSS - Cascading Style Sheets

O que é o CSS?

O CSS - Folha de estilo em cascata é uma tecnologia para definir a aparência visual de um documento HTML. Enquanto o HTML possui tags que estruturam o documento, o CSS diz como esse conteúdo deverá ser exibido.

O que são regras CSS?

Toda instrução de estilo em CSS é chamada de regra, sendo que uma regra é composta por:

  • Seletor: Indica quais elementos HTML serão estilizados e podem ser de três tipos: Id, Tipo e Classe.
  • Bloco de Declarações: Nada mais é que um par de propriedade: valor.
  • Propriedade: É o aspecto visual que deverá ser controlado, tais como: cor de fonte, margem, cor de fundo, tipo de fonte etc.
  • Valor: É o atributo da propriedade que queremos modificar.

Seletor de Tipo

O seletor de tipo seleciona todos os elementos de uma determinada tag HTML na página. A sintax desse seletor é o nome da tag HTML, sem nenhum prefixo. Neste exemplo, estamos usando vários seletores de tipo.
