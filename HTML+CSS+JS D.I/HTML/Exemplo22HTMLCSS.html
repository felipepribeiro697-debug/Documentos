Formas de Vinculação do CSS

Quando falamos de formas de vinculação de arquivos CSS, estamos nos referindo sobre de que forma vamos criar e vincular os arquivos nos documentos HTML. Existem basicamente duas formas, a vinculação interna (menos recomendada) e a vinculação externa (mais recomendada).

Neste exemplo, estamos usando a vinculação externa, ou seja, criamos um arquivo CSS externo e vinculamos ele ao documento. Essa abordagem é mais recomendada pois facilita dentre outras coisas a manutenção do arquivo e da própria página.
