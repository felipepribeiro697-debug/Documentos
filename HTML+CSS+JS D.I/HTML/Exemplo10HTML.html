Conceitos Gerais

A Web Semântica é uma extensão da Web atual, idealizada pelo Consórcio World Wide Web (W3C), onde as informações ganham significado bem definido, permitindo que máquinas processem, interpretem e conectem dados de forma inteligente. Ela transforma documentos em "dados conectados", facilitando buscas automáticas, interoperabilidade e compreensão do contexto por agentes de software.

Características

  • Dados Estruturados e Interligados (Linked Data):Utiliza grafos para conectar informações de diferentes fontes, transformando a web de um sistema de documentos em um grande banco de dados distribuído.
  • Significado (Semântica): O conteúdo não é apenas exibido, mas descrito com metadados que explicam seu contexto, permitindo que as máquinas "entendam" o que o dado representa.
  • Inferência e Raciocínio Lógico: Capacidade de deduzir novas informações a partir de dados existentes, usando regras definidas (como RDF e OWL) para chegar a conclusões automáticas.
  • Representação por Ontologias: Utiliza linguagens como OWL (Ontology Web Language) para definir conceitos, classes e as relações entre eles, garantindo uma estrutura coerente e compartilhada do conhecimento.
  • Identificação com URIs: Cada recurso informacional é identificado de forma única e padronizada (Uniform Resource Identifier), facilitando sua localização e referenciamento na web.
  • Descentralização: Não depende de uma autoridade centralizada; permite que diferentes comunidades criem e interconectem seus próprios modelos de dados.

Aplicações

  • Integração de Dados e Linked Data: Conecta bases de dados independentes e diversas, transformando a web em uma base de dados distribuída, facilitando a interoperabilidade entre sistemas.
  • Busca Semântica e Motores de Conhecimento: Ferramentas como o Hakia.com e mecanismos de busca analisam o contexto e significado da pesquisa em vez de apenas palavras-chave, resultando em respostas mais precisas.
  • Internet das Coisas (IoT): A semântica é utilizada para permitir que dispositivos e sensores heterogêneos compartilhem informações e colaborem, estabelecendo um entendimento comum de dados.
  • Gestão de Conhecimento Organizacional: Uso de ontologias e padronização de metadados para classificar, armazenar e recuperar conhecimento corporativo de forma mais eficiente, facilitando a tomada de decisão.
  • Agentes Inteligentes e Assistentes Virtuais: Sistemas que se adaptam e aprendem com o comportamento do usuário, como o Siri.com (exemplo citado em 2012).
  • Processamento de Linguagem Natural (PLN): Classificação de sentimentos e análise de feedback de clientes em grande escala, como no YouTube.
